# UAE Match 项目介绍

## 项目概述

**UAE Match** 是一个基于人工智能的智能相亲匹配平台。这个项目结合了先进的AI技术（GPT-4o-mini）和向量相似度搜索算法，为在阿联酋地区的用户提供个性化的匹配推荐服务。

### 核心定位
- **目标用户**：在阿联酋地区寻找伴侣的用户
- **核心价值**：通过AI技术提供更精准、更个性化的匹配推荐
- **技术特色**：AI驱动的匹配算法 + 智能对话助手

---

## 项目架构

### 技术栈

#### 后端（Backend）
- **框架**：Python + FastAPI（现代化、高性能的异步Web框架）
- **数据库**：SQLite（开发环境）/ PostgreSQL（生产环境推荐）
- **AI服务**：OpenAI GPT-4o-mini（成本优化、性能优秀）
- **向量搜索**：text-embedding-3-small（用户特征向量化）
- **实时通信**：WebSocket（即时聊天功能）
- **其他核心库**：
  - SQLAlchemy（ORM数据库操作）
  - JWT认证（安全的用户身份验证）
  - 图像处理（Pillow）

#### 前端（Frontend）
- **框架**：React 18 + TypeScript（类型安全、组件化开发）
- **构建工具**：Vite（极速开发体验）
- **UI组件库**：Radix UI + Tailwind CSS（现代化、可访问的设计系统）
- **状态管理**：React Hooks
- **表单处理**：React Hook Form

## 核心功能模块

### 1. 🤖 AI智能匹配推荐系统

**工作原理**：
```
用户资料 → AI向量化 → 向量数据库 → 相似度搜索 → 智能筛选 → 个性化推荐
```

**技术细节**：
- 将用户的个人信息（职业、兴趣、价值观等）转换为1536维向量
- 使用余弦相似度算法找到最匹配的用户
- 结合硬性条件过滤（年龄、身高、城市等）
- 最后用AI为每个推荐对象生成个性化的推荐理由

**为什么这样做**：
传统匹配软件只能根据简单的标签（如"喜欢旅行"）进行匹配。而我们的AI系统能理解更深层的语义，比如"喜欢旅行"和"热爱探索"虽然表述不同，但能识别为相似的兴趣。

---

### 2. 💬 AI红娘助手（"小缘"）

**功能特点**：
- **注册引导**：帮助新用户完成资料填写，通过对话的方式收集信息，体验更自然
- **日常咨询**：随时解答用户关于匹配、聊天技巧等问题
- **VIP主动关怀**：定期主动关心用户，询问匹配进展，提供建议
- **长期记忆**：AI会记住与用户的每一次对话，通过滚动摘要技术，不断学习用户的偏好

**技术亮点**：
- 使用滚动摘要技术（Rolling Summary），每次对话后自动压缩为500-800字的摘要
- 这样AI既不会"忘记"重要信息，也不会因为历史对话太长而变得缓慢
- 智能防打扰机制：如果用户正在活跃聊天，AI不会打扰

## 技术亮点与创新

### 1. 混合匹配算法
- **硬性过滤**（SQL）：年龄、身高、地理位置等明确条件
- **软性匹配**（向量搜索）：性格、兴趣、价值观等模糊条件
- 结合两者，既保证准确性，又提高匹配质量

### 2. 成本优化的AI使用策略
- 全部使用GPT-4o-mini（成本比GPT-4低很多，但性能优秀）
- 向量化使用text-embedding-3-small（成本极低）
- 预计1000普通用户+100VIP用户，月度成本约$212-257（在可控范围内）

### 3. 智能缓存机制
- 匹配分析结果缓存（避免重复计算）
- 用户向量嵌入缓存
- 自动清理过期缓存

### 4. 异步任务调度
- 定时清理缓存
- VIP用户主动关怀（每周二、五晚上8点）
- 后台任务不影响主服务性能

## 使用场景示例

### 用户注册流程
1. 用户访问网站，看到精美的登录页
2. 点击"开始"后，进入AI引导流程
3. AI助手"小缘"通过对话方式收集用户信息：
   - "你好！我是小缘，很高兴认识你。能告诉我你的名字吗？"
   - "你平时喜欢做什么？有什么兴趣爱好吗？"
   - "你对另一半有什么期待吗？"
4. 信息收集完成后，系统自动生成用户向量嵌入
5. 资料提交审核，审核通过后即可使用

### 每日推荐流程
1. 用户打开"每日推荐"页面
2. 系统基于用户的向量嵌入，在数据库中找到最相似的5位用户
3. AI为每位推荐对象生成个性化推荐理由：
   - "小缘推荐：张先生是一位热爱旅行的IT工程师，与你喜欢探索新事物的性格很匹配..."
4. 用户看到推荐后，可以点击"喜欢"或"跳过"
5. 如果双方都"喜欢"，系统自动创建匹配，双方可以开始聊天

### VIP关怀流程
1. 系统检测到用户匹配成功已经24小时
2. 触发VIP关怀任务
3. AI主动发送消息：
   - "小缘关心：看你昨天匹配成功了，聊得怎么样？有什么需要我帮助的吗？"
4. 用户回复后，AI提供个性化建议
