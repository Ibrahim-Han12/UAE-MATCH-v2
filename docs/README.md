## 术语约定

| 缩写 | 全称 | 含义 |
|---|---|---|
| BRD | Business Requirements Document | 业务需求文档 |
| HLD | High-Level Design | 高层（概要）设计 |
| LLD | Low-Level Design | 详细（底层）设计 |
| VIP | - | 付费高级会员 |
| 小缘 | - | 平台的 AI 红娘助手（AI 角色名） |

## 阅读建议

- 想快速了解"这是个什么产品" → 读 **BRD**
- 想了解"系统怎么搭起来的" → 读 **HLD**
- 想了解"用户看到什么、怎么用" → 读 **UI-UX**
- 想动手改代码 / 对接接口 → 读 **LLD**

## 当前版本状态（重要）

v1.0 是一个**功能骨架已完整、可本地运行**的版本，但距离商业上线仍有明确缺口（支付未接真实网关、通知系统为桩实现、部分前端为演示态等）。各文档末尾均标注了"现状与缺口"，后续优化以此为准。
